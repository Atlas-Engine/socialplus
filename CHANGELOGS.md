## SocialPlus v1.11 — Changelog

### English
✨ Improvements  
• Reworked right-click friend menu with proper Blizzard-style alignment and dynamic width.  
• Friend header text now left-aligned, anchored cleanly, and scales for long names.  
• Menu auto-expands while respecting minimum width rules.  
• Added a **new global collapse/expand button** on the *left side of the Friends tab*, toggling all groups at once.  
• General (ungrouped) header is now fixed open — no +/- icon, no collapsing behavior.  
• Added **Move to Group** when a friend already belongs to a group, showing all other available groups.  
• Removed unused Target functionality and legacy helpers to eliminate taint.  
• Internal cleanup for reduced file size and improved maintainability.

### Français
✨ Améliorations  
• Menu clic-droit entièrement revu pour un alignement Blizzard et une largeur dynamique.  
• Le nom d’entête est maintenant aligné à gauche, bien ancré et compatible avec les longs noms.  
• Le menu s’ajuste automatiquement tout en respectant une largeur minimale.  
• Ajout d’un **nouveau bouton global de repli/dépli** situé *à gauche de l’onglet Amis*, basculant tous les groupes en même temps.  
• Le groupe Général reste toujours ouvert — aucun icône +/- et aucun repli possible.  
• Ajout de **Déplacer vers un groupe** lorsqu’un ami appartient déjà à un groupe, listant tous les autres groupes disponibles.  
• Suppression du code Target obsolète pour éviter tout problème de taint.  
• Nettoyage interne pour alléger le fichier et améliorer la maintenabilité.
